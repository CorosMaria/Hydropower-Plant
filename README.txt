Download the project to local storage.
Open Visual Studio Code and open the folder my-app.
Delete the yarn.lock file.
Run command yarn build in command prompt terminal. 
After the previous command is done, run yarn start. 
The page will run at localhost:3000.
To control the hydropower instalation we use the controller: http://www.science.upm.ro/~traian/web_apl/multiio/act_val_multi.html
This web page is a part of an larger system developed to control and monitor a hydropower plant.
