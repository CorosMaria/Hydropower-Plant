Download the project to local storage.

WEB PAGE - my-app folder 
1. Open Visual Studio Code and open the folder my-app.
2. Delete the yarn.lock file.
3. Run command yarn build in command prompt terminal. 
4. After the previous command is done, run yarn start. 
5. The page will run at localhost:3000.
6. To control the hydropower instalation we use the controller: http://www.science.upm.ro/~traian/web_apl/multiio/act_val_multi.html
This web page is a part of an larger system developed to control and monitor a hydropower plant.
